\# SmartDiab AI: Health Risk Assessment



SmartDiab AI is a machine learning-powered health tool designed to predict risks for \*\*Diabetes, Cardiovascular issues, and Hypertension\*\*. Built using a high-performance \*\*XGBoost\*\* model and served via a \*\*FastAPI\*\* backend, this project provides a fast and scalable way to assess health data.



\## 🚀 Features

\* \*\*Risk Prediction:\*\* Uses XGBoost for high-accuracy health risk assessment.

\* \*\*FastAPI Backend:\*\* Provides a high-speed API for seamless integration.

\* \*\*Organized Structure:\*\* Modular codebase with dedicated folders for API, ML model, and Frontend.



\## 📁 Project Structure

\* `ai-api/`: Contains the FastAPI implementation.

\* `ml-model/`: The trained XGBoost model and preprocessing scripts.

\* `backend/` \& `frontend/`: Core application logic and user interface.
